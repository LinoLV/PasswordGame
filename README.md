# **PasswordGame**
PasswordGame is an **🅾🅵🅵🅻🅸🅽🅴** game.
The goal of the game is to make a password by following the on-screen conditions. All the correct passwords entered will win!🎖️ \
**v1.0** and **v2.0** has completely different UI.
## **📝Installation**
### For Windows:
Download the .exe file from the [latest release](https://github.com/LinoLV/PasswordGame/releases/latest/) and run the game.
### For Linux or MacOS:
・The first step is to install wine from https://winehq.org/. _(Skip this step if you already have wine installed)_ \
・The second step is to download the .exe file from the [latest release](https://github.com/LinoLV/PasswordGame/releases/latest/) and run the game.
### For Android:
Coming Soon (Not supported yet)
## **💻System Requirements**
Works on almost every **Windows based PCs** with atleast 50MB of disk space. \
Also works on **Linux** and **MacOS** if you have [wine](https://www.winehq.org/) installed. \
*(Android support coming soon.)*
